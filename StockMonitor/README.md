# StockMonitor

这是一个基于 Python 的 **股价提醒系统**，支持：
- 监控一个或多个股票代码，按用户设定的条件（如价格 ≥ X、价格 ≤ Y、涨幅 ≥ Z%、量能放大等）进行检测；
- 当条件首次触发时，通过控制台打印或通过钉钉（DingTalk）机器人 Webhook 发送通知；
- 支持任务新增、删除、列出、清空、重置通知状态、启动监控；
- 支持备注字段，通知时会显示备注；
- 支持基于前一日收盘价计算涨幅预警、量能预警；
- 支持 Docker 容器化部署。

## 功能一览
- 添加监控任务：股票代码、条件类型（`ge`, `le`, `change_pct_up`, `change_pct_down`, `vol_up`）、监控频率（秒）、通知方式（console / dingtalk）、钉钉 Webhook（如选用）、备注；
- 列出所有任务，展示任务 ID、股票代码、条件、频率、通知方式、Webhook 地址、备注、是否已通知；
- 删除任务；
- 启动监控：系统每隔指定频率拉取当前数据（最新 + 前一日），判断价格变动或量能变化，首次满足条件即提醒；
- 重置所有任务的 “已发送通知” 状态；
- 清空所有任务（需要二次确认，防误操作）；
- 持久化存储任务（`tasks.json`），重启后继续监控。
